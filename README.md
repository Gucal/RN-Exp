## Proje Kurulumu

1.)Projeyi klonlayın.
2.)Proje ana dizininde `npm install` ya da `yarn install` komutunu çalıştırın.
3.) Ardından ana dizine gelerek `npx react-native run-ios` ya da `npx react-native run-android` komutu ile projeyi çalıştırın.
